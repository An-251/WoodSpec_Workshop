# Nhật ký thay đổi WoodSpec

## Cập nhật nội dung

- Chuẩn hóa giao diện sang tiếng Việt.
- Đổi dữ liệu dài sang `src/data/reference`.
- Làm rõ các trang: trang chủ, bảng điều khiển, cấu hình, bảng thông số và báo giá.
- Bỏ các nhãn thử nghiệm khỏi nội dung hiển thị.

## Cập nhật kỹ thuật

- Bản vẽ hai chiều dùng chung `Cabinet2DSvg`.
- Trang sản phẩm dùng cùng hệ bản vẽ với trang cấu hình.
- Báo giá tham khảo thay đổi theo giá ước tính của cấu hình hiện tại.
- Trạng thái cấu hình được lưu bằng Zustand.
